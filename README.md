#  Currency Converter - Multi-Currency & Multi-Language
# Link: https://waehrungsumrechner-app.netlify.app/


---

##  Überblick

Ein **responsiver Währungsrechner** mit folgenden Highlights:
- 🇺🇳 **21 internationale Währungen** mit echten Flaggen
-  **5 Sprachen** (Deutsch, Englisch, Italienisch, Türkisch, Arabisch)
-  **Live-Wechselkurse** über eine kostenlose API
-  **100% Responsive** – perfekt für Handy, Tablet und Desktop
-  **Sofortige Umrechnung** beim Tippen

---

##  Features

| Feature | Beschreibung |
|---------|-------------|
|  **Echte Flaggen** | Jede Währung hat ihre Landesflagge (EU-Flagge für Euro) |
|  **Mehrsprachig** | 5 Sprachen mit automatischer RTL-Unterstützung für Arabisch |
|  **Live-Kurse** | Echtzeit-Wechselkurse von ExchangeRate-API |
|  **Responsive** | Optimiert für alle Bildschirmgrößen |
|  **Modernes Design** | Gradienten, Animationen und Hover-Effekte |
|  **Sofortige Konvertierung** | Ergebnis aktualisiert sich beim Tippen |
